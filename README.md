# 🌌 Antigravity OS

> **The Behavioral OS for High-Performance AI Engineering.**

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

**Antigravity OS** is a set of modular guidelines, senior-level engineering skills, and a premium visualization dashboard designed to push AI agents (like Claude Code, Cursor, and Antigravity) beyond their default limits. Inspired by Andrej Karpathy's observations on LLM coding pitfalls, this project provides the **Behavioral Layer** required for financial-grade, aesthetic software development.

---

## ✨ Key Features

- **🛡️ The Golden Rules**: A core behavioral contract (`ANTIGRAVITY.md`) that enforces surgical engineering and **Zero-Leakage** security.
- **🧩 Modular Skill System**: Injected discipline modules for CSS Layouts, Backend Architecture, and **Security Protocols**.
- **🖥️ Premium Dashboard**: A glassmorphic, React-inspired UI to visualize active rules and system status.
- **🎯 Goal-Driven Execution**: Framework for transforming vague requests into verifiable, testable engineering outcomes.

---

## 🚀 Quick Start

No cloning necessary. Just run this command inside any project where you want to apply the Antigravity OS:

```bash
npx jjw3310/Antigravity_OS setup
```

### What this does:
1.  Injects **`ANTIGRAVITY.md`** (Golden Rules) into your project root.
2.  Installs modular **Skill Modules** into `_agents/skills/`.
3.  Instantly elevates your AI agent's engineering discipline.

---

### Manual Installation (Advanced)
If you prefer to browse the source or customize the core:
```bash
git clone https://github.com/jjw3310/Antigravity_OS.git
```

## 🧠 The Philosophy: Surgical Engineering
We believe that AI agents are most effective when they are constrained by senior-level discipline. Antigravity OS focuses on:
- **Minimum Churn**: Touch only what is necessary.
- **Think Before Coding**: Explicitly state assumptions before execution.
- **Wait & Verify**: Every change must be verifiable via automated checks.
